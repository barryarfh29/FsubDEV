Admin Konten & Custom Text

Pilih tombol di bawah agar halaman tidak terlalu panjang. Admin Konten sengaja ditaruh paling depan karena lebih penting untuk kontrol siapa yang boleh generate link.

Placeholder custom text: {mention}, {first_name}, {user_name}, {username}, {bot_name}, {bot_username}, {required_count}, {join_note}, {link}, {reason}, {blocked_detail}

Penjelasan
  • Admin konten hanya boleh kirim text/media ke bot untuk generate link.
  • Mereka tidak bisa ubah setting website, channel database, channel wajib join, atau start/stop bot.
  • Owner ID tidak perlu dimasukkan lagi.
  • Satu Telegram ID per baris. Bisa juga dipisah koma atau spasi.
